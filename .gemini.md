# The Council 2.0 - Gemini Context

This project is a multi-agent autonomous AI system for data analysis. It uses a hybrid approach: LLMs for reasoning and Polars for fast, deterministic data execution.

## 🛠 Tech Stack
- **Backend**: Python, FastAPI, LangGraph, Polars, Faiss, Ollama.
- **Frontend**: Next.js, TypeScript, Tailwind CSS, Plotly.
- **Dependency Management**: `uv` (Python), `npm` (Node.js).

## 🏗 Architectural Patterns
- **Multi-Agent Orchestration**: LangGraph manages the state and routing between specialized agents (Router, Analyst, Designer, Librarian).
- **Lazy Loading**: Engines and models are loaded only when needed to minimize boot time.
- **Deterministic Analytics**: Always prefer `QueryEngine` (Polars) for data operations over pure LLM hallucinations.

## ⚖ Coding Standards
- **Zero Bloat**: Avoid unnecessary dependencies. Use consolidated libraries.
- **Thermodynamic Refactoring**: Minimize entropy; keep functions high-density and low-complexity.
- **Explicit Imports**: Use absolute imports within the backend.

## 📂 Key Locations
- **Agents**: `backend/agents/`
- **Engines**: `backend/engines/`
- **Graph Logic**: `backend/workflow/`
- **UI Components**: `frontend/components/`
